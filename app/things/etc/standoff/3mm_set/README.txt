# Summary

I designed a M3 standoff, perfect for mounting FC's.

there are different sizes, the names describe them as following:
for example: M3_8mm_5mm_5.5mm
which means M3 thread, 8mm tall base, 5mm tall thread, 5.5mm wrench size.

I recommend to clean and cut the threads by screwing in a metal bolt/nut a few times.

recommended printer settings:
* 0.05mm
* Infill
* 10mm/s

printed on a Kodama Trinus.
